# Hardware

This folder is for microscope-camera adapter files and hardware documentation.

## Planned Files

- 3D-printable adapter model files, such as STL or STEP
- adapter version notes
- camera mounting dimensions
- microscope eyepiece or objective mount dimensions
- lighting setup photos
- calibration slide examples

## Design Requirements

The adapter should help standardize:

- camera distance
- camera angle
- field of view
- illumination geometry
- sample position

Stable imaging geometry is essential because inconsistent image quality can
cause computer vision models to fail in routine lab use.
