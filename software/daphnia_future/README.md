# Future Daphnia Workflow

This folder records ideas that may transfer from the *C. elegans* pilot to a
future *Daphnia* behavior and water quality project.

## Transferable Parts

- image and video acquisition protocol
- annotation workflow
- object detection evaluation
- multi-object tracking logic
- behavior feature extraction
- validation against manual review

## Likely Daphnia Features

- swimming speed
- turning frequency
- immobility
- vertical distribution
- abnormal movement
- group-level behavior
- response to water quality stress

The biological model will need separate data and training, but the engineering
workflow can be reused.
